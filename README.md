# Magic Calculator
Final project for Fcode C Training.

**Magic Calculator** is a C program that can solve all operations by one line.

# Requirement
- GCC 5.4.0 and up.

# Installation
```
git clone https://github.com/bkdev98/magic-calculator
cd magic-calculator
gcc-5.4.0 -o mcalc main.c
./mcalc
```
